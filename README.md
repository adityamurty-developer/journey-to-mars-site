# Journey to Mars 🚀


**Journey to Mars** is an interactive, scroll-driven web experience that takes users on a voyage from Earth to the Red Planet. Built using **HTML, CSS, and vanilla JavaScript** (no frameworks), it combines animations, hover interactions, and interactive storytelling to create an immersive exploration of space.


## Features

- **Hero Section**: Animated rocket with a "Begin Journey" button.
- **Launchpad**: Countdown timer and liftoff animation.
- **Space Section**: Distance counter, floating planets with hover tooltips, scroll-based interaction.
- **Mars Landing**: Rover landing animation with status updates.
- **Surface Exploration**: Rover movement across Mars terrain and fixed glowing badges revealing info about water, atmosphere, biology, geology, and temperature.
- **Mission Complete**: Celebration animation and return-to-Earth functionality.


## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
- GSAP for smooth animations, Canva for visual assets
