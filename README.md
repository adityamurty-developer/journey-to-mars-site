# Journey to Mars 🚀


**Journey to Mars** is an interactive, scroll-driven web experience built with **HTML, CSS, and vanilla JavaScript**, created with the help of AI. Users explore the journey from Earth to Mars with animations, hover interactions, and immersive storytelling.


## Sections Overview

1. **Hero** – Animated rocket with "Begin Journey" button to start the adventure.  
2. **Launchpad** – Countdown timer and rocket liftoff animation.  
3. **Space** – Scroll-driven distance counter with floating planets; hover tooltips reveal info.  
4. **Mars Landing** – Rover finds landing spot and touches down with animated status updates.  
5. **Surface Exploration** – Rover moves across Mars; fixed glowing badges display information on water, atmosphere, biology, geology, and temperature.  
6. **Mission Complete** – Celebration animation with a “Return to Earth” button to restart the journey.


## Technologies Used

- HTML5  
- CSS3  
- Vanilla JavaScript  
- GSAP for smooth animations, Canva for visual assets
